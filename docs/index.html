Tessera

A keyboard-driven tiling window manager for macOS — tmux/Zellij workflows for any app.

macOS 14+HomebrewMIT

Install

brew tap pa/tessera https://github.com/pa/tessera
brew install --HEAD tessera
brew services start tessera

After installing, grant System Settings → Privacy & Security → Accessibility → Tessera. The grant survives every brew upgrade. Keep Stage Manager off.

What it does

Tessera is a menu-bar agent (look for the ▚ glyph — the pill also shows your tab position and active mode). It puppets other apps' windows through the macOS Accessibility API to give you a terminal-multiplexer workflow across your whole desktop.

BSP tiling

Split any focused window right or down, with configurable padding.

Virtual tabs

Independent workspaces; a tab can stack windows in monocle mode.

Modal keys

Zellij-style Pane / Tab / Resize modes with a live, context-aware hint bar.

Command palette

Fuzzy-find and launch apps or focus any window.

Workspace navigator

A searchable tree of tabs, panes, and windows with app icons.

Floating & zoom

Pop a window out to float, or zoom a pane to fill the screen.

Organize by app

One command lays every open window out as one tab per app.

Session restore

Your layout comes back after a restart — windows re-slotted, no relaunch.

Stubborn apps

Toggles AXEnhancedUserInterface so Chromium/Electron apps actually tile.

Keyboard reference

The default prefix is ⌃⌥⌘ (Control-Option-Command). Everything is rebindable in Settings (⌘, or the menu bar).

KeyAction

KeyAction
nNew tab
h / lPrevious / next tab
⇧h / ⇧lMove window to previous / next tab

KeyAction

In any mode the menu-bar hint bar shows only the keys that apply right now. ⏎ / esc exits a mode.

Usage & configuration

Getting started

After granting Accessibility, Tessera automatically lays your open windows out as one tab per app (a multi-window app's windows are stacked so you cycle them with n/p in Pane mode). Re-run this anytime from the menu bar → Organize Windows by App. From there, split, move windows between tabs, and float as you like.

Configuring shortcuts

Open Settings — ⌘, while Tessera is active, or the menu-bar ▚ menu → Settings… — and pick the Hotkeys tab. Click any shortcut and press the new chord to rebind it (at least one modifier required; esc cancels). Reset to Defaults restores the built-in set. Changes apply immediately.

Tile padding

In Settings → General, drag the Tile padding slider — it's a percentage of screen width applied to both the outer margin and the gaps between panes, live. The same tab also shows the app version.

Accessibility & menu bar

Tessera runs as a menu-bar agent — the ▚ pill shows your current tab position and active mode. The menu holds the command palette, workspace navigator, Organize Windows by App, Settings, and the Accessibility grant shortcut. Keep Stage Manager off.

▚ Tessera — built with Swift + AppKit + the Accessibility API.
