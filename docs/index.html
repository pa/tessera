Tessera

A keyboard-driven tiling window manager for macOS — tmux/Zellij workflows for any app.

macOS 14+HomebrewNo Apple Developer IDMIT

Install

brew tap pa/tessera https://github.com/pa/tessera
brew install --HEAD tessera
brew services start tessera

A single Swift binary built from source — no notarization, no quarantine. Then grant System Settings → Privacy & Security → Accessibility → Tessera. The grant survives every brew upgrade. Keep Stage Manager off.

What it does

Tessera is a menu-bar agent (look for the ▚ glyph — the pill also shows your tab position and active mode). It puppets other apps' windows through the macOS Accessibility API to give you a terminal-multiplexer workflow across your whole desktop.

BSP tiling

Split any focused window right or down, with configurable padding.

Virtual tabs

Independent workspaces; a tab can stack windows in monocle mode.

Modal keys

Zellij-style Pane / Tab / Resize modes with a live, context-aware hint bar.

Command palette

Fuzzy-find and launch apps or focus any window.

Workspace navigator

A searchable tree of tabs, panes, and windows with app icons.

Floating & zoom

Pop a window out to float, or zoom a pane to fill the screen.

Organize by app

One command lays every open window out as one tab per app.

Session restore

Your layout comes back after a restart — windows re-slotted, no relaunch.

Stubborn apps

Toggles AXEnhancedUserInterface so Chromium/Electron apps actually tile.

Keyboard reference

The default prefix is ⌃⌥⌘ (Control-Option-Command). Everything is rebindable in Settings (⌘, or the menu bar).

Global shortcuts

ShortcutAction
⌃⌥⌘R / ⌃⌥⌘DSplit focused window right / down
⌃⌥⌘ H J K LFocus left / down / up / right
⌃⌥⌘⇧ H J K LMove window between panes
⌃⌥⌘T · ⌃⌥⌘] / ⌃⌥⌘[New tab · next / previous tab
⌃⌥⌘⇧PCommand palette
⌃⌥⌘⇧WWorkspace navigator
⌃⌥⌘⌫Reset tiling (un-manage everything)

Pane mode — ⌃P

KeyAction
r / dSplit right / down
h j k lFocus pane (or move a floating window)
⇧ + hjklSwap window with neighbour
n / pCycle focus through all windows
fFull-screen (zoom) the pane
wAttach / float / re-tile the focused window
sToggle stacked (monocle) layout
cChange the pane's window (palette)

Tab mode — ⌃T

KeyAction
nNew tab
h / lPrevious / next tab
⇧h / ⇧lMove window to previous / next tab
m then digits, ⏎Move window to tab # (a number past the end creates it)

Resize mode — ⌃R

KeyAction
h / lNarrower / wider
k / jTaller / shorter

In any mode, the menu-bar hint bar shows only the keys that apply right now. ⏎/esc exits a mode.

Settings

Open with ⌘, (when Tessera is active) or the menu bar:

  • General — tile padding (as a % of screen width), app version.
  • Hotkeys — rebind any command; Reset to Defaults.
